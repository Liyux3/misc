various projects from high school and university

### [poker-24-game](poker-24-game/)
4-player real-time card game where you race to form arithmetic expressions that equal 24. Three-phase architecture with RMI service layer, Swing client with MySQL persistence, and JMS pub/sub for real-time multiplayer matchmaking.

`Java` `RMI` `JMS` `Swing` `MySQL` `distributed-systems`

![demo](poker-24-game/demo.gif)

### [parallel-jacobi](parallel-jacobi/)
Multi-threaded Jacobi iterative solver for the 2D steady-state heat equation. Row-partitioned workload with pthread barrier synchronization, benchmarked across thread counts and grid sizes.

`C` `pthreads` `barrier-sync` `PDE` `HPC`

![demo](parallel-jacobi/demo.gif)

### [unix-shell](unix-shell/)
Custom Unix shell supporting multi-stage pipeline execution, async signal handling, background job control, and live `/proc` filesystem monitoring.

`C` `POSIX` `fork/exec` `pipe` `signal-handling`

![demo](unix-shell/demo.gif)

### [lasthope-rpg](lasthope-rpg/)
Terminal RPG set in a zombie-infested HKU campus. Turn-based combat with dodge rolls and critical hits, WASD map navigation, inventory and shop system with LabTokens currency. ~1800 lines C++.

`C++11` `smart-pointers` `ASCII-maps` `turn-based-combat`

![demo](lasthope-rpg/assets/demo.gif)

### [cpp-exercises](cpp-exercises/)
Sorting, recursion, linked lists, and other practice problems from intro C++.

### [earthquake-patterns](earthquake-patterns/)
Density-based spatial clustering on USGS earthquake catalog with rolling z-score temporal anomaly detection per seismic zone. Identifies earthquake swarms and unusual quiet periods across tectonic boundaries.

`DBSCAN` `anomaly-detection` `geospatial` `seismology`

![map](earthquake-patterns/assets/earthquake_map.png)

### [aqi-forecast](aqi-forecast/)
Next-day air quality prediction pipeline for Qingdao using engineered lag, rolling, and meteorological interaction features. Prediction intervals from tree-level variance estimation.

`Random-Forest` `time-series` `feature-engineering` `environmental-data`

![dashboard](aqi-forecast/assets/dashboard.png)
